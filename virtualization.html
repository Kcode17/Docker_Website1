What is Virtualization?

   

Virtualization is a technology useful for IT services, where it helps to extract the full potential of a physical machine by distributing its capabilities among many users and environments.

 
 

Imagine, we have 3 servers - mail server, web server, legacy applications server. Each server is being used about 30% of their full capacity. Traditionally keeping the legacy applications in a different server makes sense as they are important.But it was not easy to give 1 server multiple important tasks to run. But with virtualization, there can be two unique virtual servers created from just using the one mail server, and they can handle tasks independently.Hardware is the same, but is being used more efficiently.

 
 

After using virtualization, servers can be used till their maximum potential. This reduces the no.of servers being used. And the un-used servers could be re-used or retired to reduce cost.
